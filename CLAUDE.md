# 3D Assets Viewer

Visualiseur d'assets 3D façon explorateur Windows, en **Godot 4.7**. Détails : voir `README.md`.

## Commits & versioning

Ce dépôt utilise le **Conventional Commits**. La version SemVer, le tag `vX.Y.Z`, le CHANGELOG et
la GitHub Release sont produits **automatiquement** par la pipeline (`release-please` +
`.github/workflows/release.yml`) — ne jamais choisir ni poser de tag de version à la main.

Rédige chaque commit selon la règle complète :
@.claude/rules/conventional-commits.md
